Hydroponics

We started with some more serious projects, involving real deals with companies and actually creating products. This is a blog about this period. A lot of info about the product and the journey can be found there. We had to create an smart object that gave back to nature, so we build a self watering plant and used a method called “hydroponics” to feed the plant. With an arduino and our own creativity we made plans with a company to help us build a prototype. We actually made it pretty far, but because some internal conflicts within the team we stopped working on our idea. I learned a lot from this project tho, from business models to properly working in teams.

VR Trip

Now i’m working with a team to create a VR app. We are using unity and google VR to create a platform where people can walk around in a room and experience travel destinations before they actually travel there. We want to place this “room” in an an airport to release stress travelers experience on the airport and get them in the “vacation mood”. You can follow the project here: https://github.com/guuse/Virtual-Trip .

Luchtsingel

I also came in contact with using the world to create new things. We found an old bridge in Rotterdam called “De luchtsingel”. It felt rotten and abandoned, but it was in the center of Rotterdam in a beautiful place. A website with information can be found here: Luchtsingel website (the website is dutch). It contains information about the bridge and our concept. We wanted to create a permanent livestream in which people at the bridge could make music and record it there. Their recorded clips could be found at home and the user could download them. We didn’t make this project reality, but we had everything working (some parts can be found on the website).
